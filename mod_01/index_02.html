Мы можем добавлять стили к нашему содержимому. Для этого в теге head - надо добавить новый тег style

Описание в файле.

Для того чтобы задать правило стиля можно обратится прямо по имени тега. Допустим тег p - тогда свойства описаные таким образом, передадутся всем тегам в нашем файле

Если хотим применить стиль к конкретному тегу можем задать тегу атрибут.

Атрибут в некоторй степени выделяет тег от всех остальных.

Также есть атрибут id - это уникальное имя для нашего тега. Выделяющее его среди всех прочих тегов.
